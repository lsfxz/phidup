phidup

phidup is a tool to find duplicate video files by comparing “perceptual hashes”, utilizing the pHash library. It's ~~totally ripped from~~ inspired by the pHash gem.

Be warned: This is in no way finished or stable, and my first attempt to actually code something useful, so while I welcome criticism and advice, you should not expect everything to “just work”.

Installation

phidup requires libpHash, which seems to be abandoned. Unfortunately it uses deprecated calls to ffmpeg, which makes it quite cumbersome to build, especially as I wasn't even able to get it to honor the explicitly given location of an older ffmpeg on Arch Linux which is rather annoying. Anyway, there's a version of libpHash which has some bug fixes, so you can try to build that. I intend to sit down with libpHash and get it to work with a more recent ffmpeg, but that might still take a while.

Once you have libpHash installed, you can run

$ bundle install
$ gem build ./phidup.gemspec

And install it with:

$ gem install ./phidup-0.3.0.gem

Usage

phidup provides the phidup command, which provides (some help) via -h, until I properly describe the usage here.
